QUICK REFERENCE
================================================================================

Key Macros:
  POLYCALL_EXPORT       -> Public function/data (exported in DLL)
  POLYCALL_LOCAL        -> Private function/data (hidden from DLL)
  POLYCALL_CALL         -> Calling convention (Windows only)
  POLYCALL_C_BEGIN/END  -> C++ extern "C" wrapper

Windows Behavior:
  Build DLL:     gcc -DPOLYCALL_DLL_EXPORT ... → __declspec(dllexport)
  Import DLL:    gcc ... (default)             → __declspec(dllimport)
  Result:        polycall.dll + polycall.lib

Unix/Linux/macOS Behavior:
  Build:         gcc -fvisibility=hidden ... → __attribute__((visibility("default")))
  Result:        libpolycall.so / libpolycall.dylib
  Private:       POLYCALL_LOCAL              → __attribute__((visibility("hidden")))

Build Commands:
  make              # Build shared library (debug)
  make release      # Build release shared library
  make static       # Build static library
  make dll          # Build Windows DLL
  make clean        # Clean artifacts
  make install      # Install on Unix/Linux
  make help         # Show all targets
